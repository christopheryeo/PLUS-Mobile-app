# PLUS Mobile App - React Native

A comprehensive mobile application for Malaysia's PLUS Highway Authority, featuring journey planning, live traffic feeds, emergency services, and rewards management.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- Xcode (for iOS)
- Android Studio (for Android)

### Installation

```bash
# Install dependencies
npm install

# iOS only - Install pods
cd ios
bundle install
bundle exec pod install
cd ..

# Start Metro bundler
npm start

# Run on iOS (in another terminal)
npx react-native run-ios

# Run on Android (in another terminal)
npx react-native run-android
```

## 📱 Features

### Core Screens
- **Home Dashboard** - Quick access to all features

- **Live CCTV** - Real-time traffic camera feeds
- **Emergency SOS** - Quick emergency contact
